# 🛋 配套工具

引入了多个业界流行的 Linter 作为规范文档的配套工具，并根据规范内容定制了对应的规则包，它们包括：

规范 Lint 工具 NPM 包
JavaScript 编码规范
TypeScript 编码规范
Node 编码规范		 ESLint (opens new window) cyanic-eslint-config (opens new window)
CSS 编码规范 stylelint (opens new window) cyanic-stylelint-config (opens new window)
Git 规范 commitlint (opens new window) cyanic-commitlint-config (opens new window)
文档规范 markdownlint (opens new window) cyanic-markdownlint-config (opens new window)

cyanic-lint (opens new window) 收敛屏蔽了上述依赖和配置细节，提供简单的 CLI 和 Node.js API，让项目能够一键接入、一键扫描、一键修复、一键升级，并为项目配置 git commit 卡口，降低项目接入规范的成本。

您可以使用cyanic-lint (opens new window) 方便地为项目接入全部规范。
